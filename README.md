# 🛒 UrbanCart - Full Stack E-Commerce Website

##  Project Overview

UrbanCart is a modern full stack e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript.
The project provides a complete online shopping experience with authentication, product management, cart functionality, and checkout system.

---

##  Features

### 👤 User Authentication

* User Registration
* User Login & Logout
* Password Hashing
* Password Strength Checker
* Show/Hide Password Feature
* Session Management

### 🛍 Product Management

* Dynamic Product Display
* Responsive Product Cards
* Product Search Functionality

### 🛒 Cart System

* Add to Cart
* Remove from Cart
* Quantity Increase/Decrease
* Dynamic Total Price Calculation

### 💳 Checkout System

* Delivery Address Form
* Phone Number Input
* Order Placement
* Order Success Alert
* Automatic Cart Clearing

### 🎨 UI Features

* Responsive Design
* Modern Glassmorphism Authentication Pages
* Animated Homepage
* Gradient Effects
* Hover Animations

## ⚙️ Installation Steps

1. Install XAMPP
2. Move project folder into:

   ```text
   C:\xampp\htdocs
   ```
3. Start Apache and MySQL from XAMPP
4. Open phpMyAdmin
5. Create database:

   ```text
   ecommerce_db
   ```
6. Import/create required tables
7. Open browser:

   ```text
   http://localhost/ecommerce_project
   ```
